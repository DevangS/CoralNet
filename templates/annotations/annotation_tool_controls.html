Choosing a point to label
  • Click a text field
  • Shift-click on image
  • ↓ and ↑ : next/previous
  • . : first unannotated point
Labeling the chosen point
  • Click a label button
  • Type a label code in the field
  • Enter Return : accept a machine-suggested label
  • Ctrl+← → ↑ ↓ : navigate the label button grid
Selecting multiple points
  • Ctrl-click Cmd-click on image
  • Click the numbers next to the fields
  • , , and
Image controls
  • Left-click on image, Shift+↑ : zoom in
  • Right-click Ctrl-click on image, Shift+↓ : zoom out
  • , , and : zoom buttons
  • , , and : point display buttons
Navigate to another image
  • g then n : next image needing annotation
  • g then b : back in history
  • g then f : forward in history
Miscellaneous
  • Esc : get out of a label field
  • Ctrl+S Cmd+S : save progress
  • ? : open this controls dialog
  • Esc : close this controls dialog
