Annotations needs to be provided in a .csv format as follows:

  • imagename, row, col, label
  • ...

where imagename is the original name of the image file including file-extension; row and col are the row and column integer values where you want the new annotations; and label is the label-code of a label in your label-set. For example:

  • survey_image.jpg, 120, 330, acrop
  • survey_image.jpg, 120, 300, Favia
  • another_survey_image.jpg, 120, 300, acrop

Will add 2 point annotations to survey_image.jpg and 1 to another_survey_image.jpg.

TIP! If you are having problems with the csv file formatting, try using another text editor. We warmly recommend Sublime, it's really really nice.
